# 실시간 도로 교통 데이터 분석 및 시각화 시스템 (Capstone)

## 1. 프로젝트 개요
교차로 CCTV 영상을 활용해 차량 흐름을 실시간으로 분석하고 그 결과를 데이터로 시각화하는 교통 분석 시스템을 개발한 캡스톤 프로젝트입니다. 단순히 분석 결과를 한 번 보여주는 데 그치지 않고 실시간 데이터가 지속적으로 활용될 수 있는 구조를 만드는 데 초점을 두었습니다.

## 2. 문제 정의
기존의 교통 관리는 사후 분석 위주로 이루어져 있어 실시간으로 변화하는 상황을 즉각적으로 파악하는 데 한계가 있었습니다. 또한 영상 데이터 처리 과정에서 지연이나 오류가 발생할 경우 분석 결과 자체에 대한 신뢰도가 떨어질 수 있다는 점이 문제라고 생각했습니다.

## 3. 시스템 구조
# CCTV 영상 입력 → 객체 인식 → 데이터 처리 → 시각화
영상이 입력된 이후 어떤 과정을 거쳐 결과로 보여지는지를 기준으로, 전체 흐름을 이해하기 쉽게 정리했습니다.

## 4. 나의 역할
- 프로젝트 발표 자료 구성
- 프로젝트 결과와 구조를 문서로 정리하여 공유

## 5. 운영 관점에서 배운 점
이 프로젝트를 진행하며 단순히 데이터를 분석하는 것보다 데이터가 중간에 끊기지 않고 안정적으로 처리되는 구조가 서비스 신뢰도에 더 큰 영향을 준다는 점을 느꼈습니다. 작은 지연이나 오류도 전체 결과에 영향을 줄 수 있다는 점에서 운영 관점에서 시스템을 바라보는 시각이 중요하다는 것을 배울 수 있었습니다.

## 6. 사용한 기술
Python, OpenCV, YOLOv8, FastAPI 등
