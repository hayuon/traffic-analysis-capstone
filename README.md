# 실시간 도로 교통 데이터 분석 및 시각화 시스템 (Capstone)

## 1. 프로젝트 개요
교차로 CCTV 영상을 기반으로 차량 흐름을 실시간으로 분석하고 이를 데이터로 시각화하는 교통 분석 시스템을 개발한 캡스톤 프로젝트입니다. 실시간 데이터 흐름을 기반으로 지속적으로 활용 가능한 분석 구조를 만드는 것을 목표로 했습니다.

## 2. 문제 정의
기존 교통 관리는 사후 분석 중심으로 이루어져 실시간 상황 파악과 지속적인 데이터 활용에 한계가 있었습니다.  
또한 데이터 처리 과정에서 지연이나 오류가 발생하면 전체 분석 결과의 신뢰도가 떨어질 수 있다는 점을 고려했습니다.

## 3. 시스템 구조
CCTV 영상 입력 → 객체 인식 → 데이터 처리 → 시각화

## 4. 나의 역할
- 프로젝트 발표 자료 구성
- 프로젝트 결과와 구조를 문서로 정리하여 공유

## 5. 운영 관점에서 배운 점
이 프로젝트를 통해 단순한 데이터 분석보다 데이터가 끊기지 않고 안정적으로 흐르는 구조가 서비스 신뢰도에 더 중요하다는 것을 배웠습니다.  
작은 지연이나 오류도 전체 시스템에 영향을 줄 수 있다는 점에서 운영 관점의 설계와 관리의 중요성을 이해하게 되었습니다.

## 6. 사용한 기술
Python, OpenCV, YOLOv8, FastAPI 등
